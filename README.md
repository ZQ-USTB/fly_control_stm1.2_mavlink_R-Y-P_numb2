# 🚀 STM32H7 飞行控制系统技术概览
> **High-Performance Flight Control System based on STM32H743**

本系统集成了工业级信号处理算法与实时多任务架构，旨在为飞行器提供极致稳定的姿态控制与航向锁定。

---

## 🛡️ 核心算法：二阶巴特沃斯滤波 (Butterworth Filter)
系统在 `algorithm/biquad_filter.c` 中实现了精密的高阶信号预处理：
- **算法架构**：基于 **Biquad (双二阶)** 结构的二阶巴特沃斯低通滤波器。
- **性能指标**：
  - **Q值锁定**：精准设定为 `0.7071`，确保通带内幅频特性最平坦。
  - **动态特性**：支持实时调整采样率与截止频率，完美兼顾信号平滑度与延迟控制。
- **实战价值**：有效抑制电机高频震动对 IMU 数据的干扰，提升姿态解算精度。

---

## 🎮 控制核心：3-DOF 串级 PID 架构
系统实现了完整的 **俯仰 (Pitch)**、**横滚 (Roll)**、**偏航 (Yaw)** 三自由度闭环控制。

### 🔄 串级控制逻辑 (Cascade PID)
采用“角度环 + 角速度环”的双环嵌套架构：
1. **外环 (Angle Loop)**：追踪目标姿态，输出期望角速度。
2. **内环 (Rate Loop)**：抑制角速度波动，输出执行器 PWM 信号。

### 🧭 增强型偏航 (Yaw) 处理
针对航向角的特殊性，在 `algorithm/pid.c` 中独立开发了 `yaw_PID_calc`：
- **✨ 核心特性**：具备 **自动过零逻辑**。当角度在 $\pm180^\circ$ 边界跳转时，算法能智能识别最短路径，避免控制震荡。
- **稳定锁头**：配合高精度陀螺仪积分，实现极佳的航向锁定效果。

---

## 📡 通信协议：Mavlink 集成
系统在 `application/mavlink.c` 中实现了 **Mavlink 2.0** 协议的移植，支持与地面站（如 QGroundControl）或上位机进行双向异步通信。

### 核心功能
- **状态同步**：定时发送 `HEARTBEAT` 心跳包，维持链路活跃。
- **地面站控制 (GCS Control)**：
  - **遥控覆盖 (RC Override)**：解析 `RC_CHANNELS_OVERRIDE` 消息，通过虚拟通道直接驱动飞行逻辑，实现地面站接管。
  - **姿态指令解析**：解析 `SET_ATTITUDE_TARGET` 消息，支持将上位机发送的**四元数 (Quaternion)** 实时转换为欧拉角指令，为自主飞行提供接口。
- **高效传输机制**：
  - **环形缓冲区 (RingBuffer)**：接收端采用循环队列管理，确保高速串口数据不丢包。
  - **中断+DMA**：利用 USART3 的中断与 DMA 特性，实现零拷贝数据接收。

## 🧩 实时架构：FreeRTOS 任务调度
系统采用多优先级抢占式内核，确保关键控制逻辑的确定性延迟。

| 任务名称 | 优先级 | 职责描述 |
| :--- | :--- | :--- |
| **INS_task** | `High` | IMU 数据采集、EKF/Mahony 姿态解算 |
| **Fly_ctrl_task** | `Normal` | 遥控逻辑解析、3-DOF PID 计算、PWM 输出 |
| **mavlink_task** | `Normal` | Mavlink 协议解析、心跳发送、数据下行同步 |
| **Default_task** | `Normal` | 系统监控与后台心跳 |

## 🛠️ 硬件与驱动支撑
- **运算核心**：利用 **STM32H743** 的 480MHz 主频与双精度 FPU，从容应对复杂的浮点运算。
- **感测系统**：原生适配 **BMI088** 工业级 IMU，具备极佳的抗震性能。
- **通信接口**：**USART3** 作为主要的外部通信链路，支持 Mavlink 与 ELRS 协议切换。
- **精准执行**：通过 **Timer3/Timer5** 输出高分辨率 PWM，实现对舵机/电机的细腻微操。


---

## 📈 调试功能
- **动态调参**：支持通过遥控器物理开关（如 F 开关）实时切换控制模式或步进调整 PID 增益。
- **中点补偿**：内置 `Bias` 逻辑，允许在飞行过程中手动微调物理中位偏差。
